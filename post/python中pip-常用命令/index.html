Python 中 pip 常用命令

安装包相关命令

安装包

pip install 是最基本的命令, 用于从官方 PyPI（Python Package Index）仓库安装指定的包.

1

pip install 包名

常用选项：

  • -i: 指定镜像源（如 中科大 镜像: https://pypi.mirrors.ustc.edu.cn/simple）
  • --upgrade: 升级已安装的包到最新版本

升级包

pip install --upgrade 用于将已经安装的包升级到最新版本.

1

pip install --upgrade 包名

安装指定版本的包

pip install 包名==版本号

一次性安装多个包

pip install -r requirements.txt

requestments.txt文件内容示例：

1
2
3

requests==2.28.1
pandas==1.5.3
numpy==1.23.0

查看信息相关命令

查看已安装包

pip list

查看包详情

pip show 包名

卸载包相关命令

卸载包

pip uninstall 包名

搜索与缓存相关命令

搜索包

pip search 关键词

清除缓存

pip cache purge

检查与调试相关命令

检查依赖关系

pip check

检查当前安装来源

pip debug

导出与备份环境

导出当前环境的包

pip freeze > requirements.txt

通过导出文件重装

pip install -r requirements.txt

镜像源配置

使用临时镜像

pip install 包名 -i https://pypi.mirrors.ustc.edu.cn/simple

永久配置镜像

  • windows: %APPDATA%\pip\pip.ini
  • Linux: ~/.pip/pip.conf

添加以下内容：

1
2

[global]
index-url = https://pypi.mirrors.ustc.edu.cn/simple
